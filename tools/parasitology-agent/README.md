# وكيل مجهر الطفيليات (MIIImageView)

يربط **كمبيوتر المجهر** (MIIImageView) مع **نظام Rare Vet LIMS** عبر الشبكة.

## كيف يعمل؟

```
MIIImageView يحفظ الصورة في المجلد
        ↓
الوكيل يراقب المجلد تلقائياً
        ↓
يرفع الصورة إلى السحابة (Render) عبر الإنترنت
        ↓
تظهر في صفحة الطفيليات والتقرير
```

> **ملاحظة:** السحابة لا تصل مباشرة لمجلد الكمبيوتر — هذا البرنامج الصغير يعمل **على كمبيوتر الطفيليات** ويرسل الصور للسيرفر.

---

## التثبيت (مرة واحدة)

### 1. تثبيت Node.js
حمّل من https://nodejs.org (إصدار 18 أو أحدث) وثبّته على **كمبيوتر الطفيليات**.

### 2. نسخ المجلد
انسخ مجلد `tools/parasitology-agent` إلى الكمبيوتر، مثلاً:
`C:\RareVet\parasitology-agent`

### 3. التثبيت
افتح PowerShell في ذلك المجلد:

```powershell
cd C:\RareVet\parasitology-agent
npm install
copy config.example.json config.json
notepad config.json
```

### 4. تعديل `config.json`

| الحقل | الوصف |
|--------|--------|
| `apiUrl` | `https://rare-vet-lims.onrender.com/api` |
| `username` / `password` | حساب المختبر |
| `watchDir` | مجلد حفظ MIIImageView (مثلاً `C:\Users\اسمك\Documents\MIIImageView`) |
| `panel` | `blood` أو `stool` (افتراضي) |

### 5. التشغيل

```powershell
npm start
```

افتح في المتصفح: **http://localhost:3920**

---

## الاستخدام اليومي

1. شغّل الوكيل (`npm start`) — يمكن إضافته لبدء التشغيل مع Windows لاحقاً
2. افتح http://localhost:3920
3. أدخل **رقم العينة** (امسح الباركود أو اكتبه)
4. اختر **دم** أو **براز**
5. من MIIImageView احفظ الصورة كالعادة → **تُرفع تلقائياً**

الصور المرفوعة تُنقل إلى مجلد فرعي `uploaded` داخل مجلد المراقبة.

---

## الشبكة

- كمبيوتر الطفيليات يحتاج **إنترنت** (Ethernet كما عندكم) للوصول إلى `rare-vet-lims.onrender.com`
- لا حاجة لفتح منافذ داخلية من السحابة نحوكم — الاتصال **من الكمبيوتر إلى السحابة** فقط

---

## استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| مجلد غير موجود | صحّح `watchDir` في config.json |
| فشل تسجيل الدخول | راجع username/password |
| عينة غير موجودة | تأكد من استقبال العينة في النظام أولاً |
| صورة لم تُرفع | تأكد من إدخال رقم العينة قبل الحفظ |

---

## تشغيل تلقائي مع Windows (اختياري)

1. أنشئ اختصاراً لـ `npm start` في مجلد الوكيل
2. ضعه في مجلد Startup: `shell:startup`
