# الطريقة السهلة — وكيل الطفيليات

## مرة واحدة فقط

1. شغّل `INSTALL-AUTOSTART.bat` — يعمل مع Windows تلقائياً
2. في MIIImageView اجعل مجلد الحفظ:
   `C:\Users\User\Desktop\صور الطفيليات`

## كل يوم (خطوتان فقط)

1. **أنشئ مجلداً** باسم رقم العينة داخل `صور الطفيليات`:
   ```
   صور الطفيليات\SMP-260623-022279
   ```
2. **احفظ الصورة** من المجهر داخل هذا المجلد → تُرفع تلقائياً

لا حاجة لفتح `localhost:3920` ولا إدخال رقم العينة يدوياً.

---

## بديل بدون وكيل (من المتصفح)

1. افتح https://rare-vet-lims.onrender.com
2. صفحة **الطفيليات** → اختر العينة
3. ارفع الصورة من الصفحة مباشرة

مناسب إذا كان المجهر بجانب شاشة تعمل LIMS.
