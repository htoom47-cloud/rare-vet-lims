تثبيت LIMS Zebra Bridge — كمبيوتر الاستقبال

جهاز المعمل: 192.168.1.102 — لا تفتح ملفاً من مسار dell.

الطريقة 1 (الأسهل):

  1. حمّل الملف:
تحميل INSTALL-FROM-LAB-NETWORK.bat
  1. انقر يمين على الملف المحمّل → تشغيل كمسؤول
  2. اترك نافذة LIMS Zebra Bridge مفتوحة
  3. افتح lims.rarevetcare.com واضغط Ctrl+F5

الطريقة 2: حزمة ZIP كاملة

تحميل reception-zebra.zip

الطريقة 3: PowerShell كمسؤول — الصق هذا السطر:

Invoke-WebRequest -Uri "http://192.168.1.102:8766/INSTALL-FROM-LAB-NETWORK.bat" -OutFile "$env:USERPROFILE\Desktop\lims-zebra-install.bat" -UseBasicParsing; Start-Process "$env:USERPROFILE\Desktop\lims-zebra-install.bat" -Verb RunAs

بعد التثبيت — اختبار طباعة: (PowerShell عادي)

$zpl = (Invoke-WebRequest "http://192.168.1.102:8766/sample-label-cbc.zpl" -UseBasicParsing).Content
$body = @{ device = @{ name = "Zebra" }; data = $zpl } | ConvertTo-Json -Compress
Invoke-RestMethod -Uri "https://127.0.0.1:9101/write" -Method Post -Body $body -ContentType "application/json"
